#### a. 声明和示例
- 算术运算：
    - 默认情况下，shell只支持整数运算
    - 运算内容：
        - 加(+)
        - 减(-)
        - 乘(*)
        - 除(/)
        - 求余数(%)
- 四则运算符号：
    - 定义：
        - $(())
    - 案例：
        - echo $((1+1))
    - 定义：
        - $[]
    - 案例：
        - echo $[10-5]
    - 定义：
        - expr
    - 案例：
        - expr 10/5
    - 定义：
        - let 
    - 案例：
        - n=1; let n+=1 等价于 let n=n+1
