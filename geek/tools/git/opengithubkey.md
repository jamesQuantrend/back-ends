### 开始打造开启大门的钥匙
- 基本讲的都是在 GitHub 上怎么操作，所以这章就将核心的东西记录一下

- 下载：
    ```
        git  clone 你的项目链接地址
    ```

- 切换到项目目录下
    ```
        cd 你的项目目录名称
    ```

- 编写代码
    ```
        略
    ```

- 提交代码
    ```
        git add . //添加到暂缓区
    ```

- 再次确认提交
    ```
        git commit -m "提交内容的文字描述"
    ```

- 进行 push 
    ```
        git push 
    ```
